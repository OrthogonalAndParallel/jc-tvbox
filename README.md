## 项目说明

* 使用Google Compose Meterial Design 设计风格的开源TVBOX
* 去除java+xml，拥抱kotlin+compose
* 适配移动设备 + TV设备
* 支持多仓聚合

**目录**
```
jc-tv-app
├── app - 主应用核心
├── gradle
├── crash - 
├── player - 多媒体播放引擎
└── quickjs - JavaScript引擎
```
